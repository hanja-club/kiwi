<h2><span id="概要(개요)"></span><span class="mw-headline" id=".E6.A6.82.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade0"><rb>概要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%AC%E7%95%BF%E9%81%93&amp;action=edit&amp;section=1" title="부분 편집: 概要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade32"><rb>京畿道</rb><rp>(</rp><rt>경기도</rt><rp>)</rp></ruby></b>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade72"><rb>道</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="地理(지리)"></span><span class="mw-headline" id=".E5.9C.B0.E7.90.86.28.EC.A7.80.EB.A6.AC.29"><ruby class="hanja grade62"><rb>地理</rb><rp>(</rp><rt>지리</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E4%BA%AC%E7%95%BF%E9%81%93&amp;action=edit&amp;section=2" title="부분 편집: 地理(지리)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade52"><rb>全體的</rb><rp>(</rp><rt>전체적</rt><rp>)</rp></ruby>으로 <a href="/wiki/%EC%84%9C%EC%9A%B8%E7%89%B9%E5%88%A5%E5%B8%82" title="서울特別市">서울<ruby class="hanja grade60"><rb>特別市</rb><rp>(</rp><rt>특별시</rt><rp>)</rp></ruby></a>를 둘러싸는 <ruby class="hanja grade42"><rb>形態</rb><rp>(</rp><rt>형태</rt><rp>)</rp></ruby>로, 서울 <ruby class="hanja grade80"><rb>外</rb><rp>(</rp><rt>외</rt><rp>)</rp></ruby>에도 <a href="/wiki/%E4%BB%81%E5%B7%9D%E5%BB%A3%E5%9F%9F%E5%B8%82" title="仁川廣域市"><ruby class="hanja grade40"><rb>仁川廣域市</rb><rp>(</rp><rt>인천광역시</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E9%BB%83%E6%B5%B7%E9%81%93" title="黃海道"><ruby class="hanja grade60"><rb>黃海道</rb><rp>(</rp><rt>황해도</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E6%B1%9F%E5%8E%9F%E9%81%93&amp;action=edit&amp;redlink=1" class="new" title="江原道 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>江原道</rb><rp>(</rp><rt>강원도</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E5%BF%A0%E6%B7%B8%E5%8D%97%E9%81%93&amp;action=edit&amp;redlink=1" class="new" title="忠淸南道 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>忠淸南道</rb><rp>(</rp><rt>충청남도</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E5%BF%A0%E6%B7%B8%E5%8C%97%E9%81%93&amp;action=edit&amp;redlink=1" class="new" title="忠淸北道 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>忠淸北道</rb><rp>(</rp><rt>충청북도</rt><rp>)</rp></ruby></a>와 <ruby class="hanja grade42"><rb>接</rb><rp>(</rp><rt>접</rt><rp>)</rp></ruby>한다.
</p>
