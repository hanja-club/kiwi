<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%97%90%EC%96%B4%ED%8C%9F&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EC%95%A0%ED%94%8C&amp;action=edit&amp;redlink=1" class="new" title="애플 (없는 문서)">애플</a>이 <a href="/index.php?title=2016%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2016年 (없는 문서)">2016<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E5%85%AC%E9%96%8B&amp;action=edit&amp;redlink=1" class="new" title="公開 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>公開</rb><rp>(</rp><rt>공개</rt><rp>)</rp></ruby></a>한 <a href="/index.php?title=%EB%B8%94%EB%A3%A8%ED%88%AC%EC%8A%A4_%EC%9D%B4%EC%96%B4%ED%8F%B0&amp;action=edit&amp;redlink=1" class="new" title="블루투스 이어폰 (없는 문서)">블루투스 이어폰</a>으로, 애플을 <b>블루투스 이어폰 <ruby class="hanja grade32"><rb>占有率</rb><rp>(</rp><rt>점유율</rt><rp>)</rp></ruby> 1<ruby class="hanja grade50"><rb>位</rb><rp>(</rp><rt>위</rt><rp>)</rp></ruby></b>로 만들어준 <a href="/index.php?title=%E5%AD%9D%E5%AD%90&amp;action=edit&amp;redlink=1" class="new" title="孝子 (없는 문서)"><ruby class="hanja grade72 unknown"><rb>孝子</rb><rp>(</rp><rt>효자</rt><rp>)</rp></ruby></a><a href="/index.php?title=%E8%A3%BD%E5%93%81&amp;action=edit&amp;redlink=1" class="new" title="製品 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby></a>이다.
<a href="/index.php?title=%E7%8F%BE%E5%9C%A8&amp;action=edit&amp;redlink=1" class="new" title="現在 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>現在</rb><rp>(</rp><rt>현재</rt><rp>)</rp></ruby></a>는 에어팟(2<a href="/index.php?title=%E4%B8%96%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="世代 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>世代</rb><rp>(</rp><rt>세대</rt><rp>)</rp></ruby></a>), 에어팟 프로가 <a href="/index.php?title=%E8%B2%A9%E8%B3%A3&amp;action=edit&amp;redlink=1" class="new" title="販賣 (없는 문서)"><ruby class="hanja grade30 unknown"><rb>販賣</rb><rp>(</rp><rt>판매</rt><rp>)</rp></ruby></a>되고 있다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%97%90%EC%96%B4%ED%8C%9F&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>에 가까이 가져가면 <a href="/index.php?title=%E8%87%AA%E5%8B%95&amp;action=edit&amp;redlink=1" class="new" title="自動 (없는 문서)"><ruby class="hanja grade72 unknown"><rb>自動</rb><rp>(</rp><rt>자동</rt><rp>)</rp></ruby></a>으로 페어링되며, <a href="/index.php?title=%E9%9F%B3%E8%81%B2&amp;action=edit&amp;redlink=1" class="new" title="音聲 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>音聲</rb><rp>(</rp><rt>음성</rt><rp>)</rp></ruby></a>으로 <a href="/index.php?title=%EC%8B%9C%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="시리 (없는 문서)">시리</a>를 <a href="/index.php?title=%E5%91%BC%E5%87%BA&amp;action=edit&amp;redlink=1" class="new" title="呼出 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>呼出</rb><rp>(</rp><rt>호출</rt><rp>)</rp></ruby></a>할 수 있는 <a href="/index.php?title=%E7%AD%89&amp;action=edit&amp;redlink=1" class="new" title="等 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%EC%95%A0%ED%94%8C&amp;action=edit&amp;redlink=1" class="new" title="애플 (없는 문서)">애플</a><a href="/index.php?title=%E8%A3%BD%E5%93%81&amp;action=edit&amp;redlink=1" class="new" title="製品 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby></a>과 <a href="/index.php?title=%E8%81%AF%E5%8B%95&amp;action=edit&amp;redlink=1" class="new" title="聯動 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>聯動</rb><rp>(</rp><rt>연동</rt><rp>)</rp></ruby></a>이 매우 잘 되어 있다.
</p><p><a href="/index.php?title=%ED%84%B0%EC%B9%98&amp;action=edit&amp;redlink=1" class="new" title="터치 (없는 문서)">터치</a> <a href="/index.php?title=%EC%84%BC%EC%84%9C&amp;action=edit&amp;redlink=1" class="new" title="센서 (없는 문서)">센서</a>가 <a href="/index.php?title=%E5%85%A7%E8%97%8F&amp;action=edit&amp;redlink=1" class="new" title="內藏 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>內藏</rb><rp>(</rp><rt>내장</rt><rp>)</rp></ruby></a>되어 <a href="/index.php?title=%E5%86%8D%E7%94%9F&amp;action=edit&amp;redlink=1" class="new" title="再生 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>再生</rb><rp>(</rp><rt>재생</rt><rp>)</rp></ruby></a>이나 <a href="/index.php?title=%E4%B8%80%E6%99%82%E5%81%9C%E6%AD%A2&amp;action=edit&amp;redlink=1" class="new" title="一時停止 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>一時停止</rb><rp>(</rp><rt>일시정지</rt><rp>)</rp></ruby></a> 같은 <a href="/index.php?title=%E6%93%8D%E4%BD%9C&amp;action=edit&amp;redlink=1" class="new" title="操作 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>操作</rb><rp>(</rp><rt>조작</rt><rp>)</rp></ruby></a>을 손쉽게 할 수 있고 유닛이 <a href="/index.php?title=%EA%B7%80&amp;action=edit&amp;redlink=1" class="new" title="귀 (없는 문서)">귀</a>에서 <a href="/index.php?title=%E9%9B%A2%E8%84%AB&amp;action=edit&amp;redlink=1" class="new" title="離脫 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>離脫</rb><rp>(</rp><rt>이탈</rt><rp>)</rp></ruby></a>할 <a href="/index.php?title=%E5%A2%83%E9%81%87&amp;action=edit&amp;redlink=1" class="new" title="境遇 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby></a> <ruby class="hanja grade50"><rb>再生</rb><rp>(</rp><rt>재생</rt><rp>)</rp></ruby>을 <ruby class="hanja grade72"><rb>自動</rb><rp>(</rp><rt>자동</rt><rp>)</rp></ruby>으로 <a href="/index.php?title=%E5%81%9C%E6%AD%A2&amp;action=edit&amp;redlink=1" class="new" title="停止 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>停止</rb><rp>(</rp><rt>정지</rt><rp>)</rp></ruby></a>해주기도 한다.
</p><p>하지만 <a href="/index.php?title=%EC%95%88%EB%93%9C%EB%A1%9C%EC%9D%B4%EB%93%9C_%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0&amp;action=edit&amp;redlink=1" class="new" title="안드로이드 스마트폰 (없는 문서)">안드로이드
