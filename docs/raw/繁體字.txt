<html><head></head><body><div class="mw-parser-output"><h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%81%E9%AB%94%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade32"><rb>繁體字</rb><rp>(</rp><rt>번체자</rt><rp>)</rp></ruby>는 <a href="/wiki/%E7%B0%A1%E9%AB%94%E5%AD%97" title="簡體字"><ruby class="hanja grade40"><rb>簡體字</rb><rp>(</rp><rt>간체자</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade52"><rb>相對</rb><rp>(</rp><rt>상대</rt><rp>)</rp></ruby>되는 말로 <a href="/wiki/%E4%B8%AD%E8%8F%AF%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9C%8B" title="中華人民共和國"><ruby class="hanja grade40"><rb>中華人民共和國</rb><rp>(</rp><rt>중화인민공화국</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby> <a href="/wiki/%E7%B0%A1%E9%AB%94%E5%AD%97" title="簡體字"><ruby class="hanja grade40"><rb>簡體字</rb><rp>(</rp><rt>간체자</rt><rp>)</rp></ruby></a> <ruby class="hanja grade60"><rb>使用地</rb><rp>(</rp><rt>사용지</rt><rp>)</rp></ruby>에서 <a href="/wiki/%E6%AD%A3%E9%AB%94%E5%AD%97" title="正體字"><ruby class="hanja grade62"><rb>正體字</rb><rp>(</rp><rt>정체자</rt><rp>)</rp></ruby></a>를 부르는 말이다.
</p>
<h2><span id="意味(의미)"></span><span class="mw-headline" id=".E6.84.8F.E5.91.B3.28.EC.9D.98.EB.AF.B8.29"><ruby class="hanja grade42"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%81%E9%AB%94%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 意味(의미)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>번거로운(<a href="/index.php?title=%E7%B9%81&amp;action=edit&amp;redlink=1" class="new" title="繁 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>繁</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby></a>) <ruby class="hanja grade40"><rb>模樣</rb><rp>(</rp><rt>모양</rt><rp>)</rp></ruby>의 글<ruby class="hanja grade70"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby>라는 뜻으로, <a href="/wiki/%E7%B0%A1%E5%8C%96%E5%AD%97" title="簡化字"><ruby class="hanja grade40"><rb>簡化字</rb><rp>(</rp><rt>간화자</rt><rp>)</rp></ruby></a>에 비해 쓰기가 <ruby class="hanja grade40"><rb>複雜</rb><rp>(</rp><rt>복잡</rt><rp>)</rp></ruby>하고 귀찮은 <ruby class="hanja grade62"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby>이라고 <ruby class="hanja grade32"><rb>解釋</rb><rp>(</rp><rt>해석</rt><rp>)</rp></ruby>할 수도 있다.
</p>
<h2><span id="繁體中文(번체중문)"></span><span class="mw-headline" id=".E7.B9.81.E9.AB.94.E4.B8.AD.E6.96.87.28.EB.B2.88.EC.B2.B4.EC.A4.91.EB.AC.B8.29"><ruby class="hanja grade32"><rb>繁體中文</rb><rp>(</rp><rt>번체중문</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E7%B9%81%E9%AB%94%E5%AD%97&amp;action=edit&amp;section=3" title="부분 편집: 繁體中文(번체중문)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>이 <ruby class="hanja grade62"><rb>字形</rb><rp>(</rp><rt>자형</rt><rp>)</rp></ruby>으로 적은 <a href="/wiki/%E4%B8%AD%E5%9C%8B%E8%AA%9E" title="中國語"><ruby class="hanja grade70"><rb>中國語</rb><rp>(</rp><rt>중국어</rt><rp>)</rp></ruby></a>를 가리키는 말이다.
