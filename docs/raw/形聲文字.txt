<h2><span id="槪要"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81"><a href="/wiki/%E6%A7%AA%E8%A6%81" title="槪要"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%85%AD%E6%9B%B8" title="六書"><ruby class="hanja grade62"><rb>六書</rb><rp>(</rp><rt>육서</rt><rp>)</rp></ruby></a>의 분류 중 하나로 의미를 나타내는 부분과 <a href="/index.php?title=%EC%86%8C%EB%A6%AC&amp;action=edit&amp;redlink=1" class="new" title="소리 (없는 문서)">소리</a>를 나타내는 부분을 합쳐서 만들어낸 글자를 가리킨다.
</p>
<h2><span id="特徵"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5"><a href="/index.php?title=%E7%89%B9%E5%BE%B5&amp;action=edit&amp;redlink=1" class="new" title="特徵 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E5%BD%A2%E8%81%B2%E6%96%87%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 特徵">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97" title="會意文字"><ruby class="hanja grade62"><rb>會意文字</rb><rp>(</rp><rt>회의문자</rt><rp>)</rp></ruby></a>와 마찬가지로 <a href="/index.php?title=%E4%B8%8A%E4%B8%8B%E5%B7%A6%E5%8F%B3&amp;action=edit&amp;redlink=1" class="new" title="上下左右 (없는 문서)"><ruby class="hanja grade72 unknown"><rb>上下左右</rb><rp>(</rp><rt>상하좌우</rt><rp>)</rp></ruby></a>나 안과 밖, 귀퉁이 등 다양한 방법으로 짜일 수 있으며, <a href="/index.php?title=%E5%BD%A2%E9%83%A8&amp;action=edit&amp;redlink=1" class="new" title="形部 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>形部</rb><rp>(</rp><rt>형부</rt><rp>)</rp></ruby></a>(의미를 나타내는 부분)을 어디에 두고 <a href="/index.php?title=%E8%81%B2%E9%83%A8&amp;action=edit&amp;redlink=1" class="new" title="聲部 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>聲部</rb><rp>(</rp><rt>성부</rt><rp>)</rp></ruby></a>(소리를 나타내는 부분)을 어디에 둔다는 <a href="/index.php?title=%E8%A6%8F%E5%89%87&amp;action=edit&amp;redlink=1" class="new" title="規則 (없는 문서)"><ruby class="hanja grade50 unknown"><rb>規則</rb><rp>(</rp><rt>규칙</rt><rp>)</rp></ruby></a>이 따로 없다. 또, <a href="/wiki/%E9%9F%B3" title="音"><ruby class="hanja grade62"><rb>音</rb><rp>(</rp><rt>음</rt><rp>)</rp></ruby></a>을 따올 때에는 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>가 만들어진 <a href="/index.php?title=%E5%8F%A4%E4%BB%A3%E4%B8%AD%E5%9C%8B%E8%AA%9E&amp;action=edit&amp;redlink=1" class="new" title="古代中國語 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>古代中國語</rb><rp>(</rp><rt>고대중국어</rt><rp>)</rp></ruby></a>의 소리에서 비슷한 <a href="/index.php?title=%E9%9F%BB&amp;action=edit&amp;redlink=1" class="new" title="韻 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>韻</rb><rp>(</rp><rt>운</rt><rp>)</rp></ruby></a>을 기반으로 하기 때문에, 다양한 짜임과 더불어 처음 보는 글자가 나왔을 때 <a href="/index.php?title=%E6%84%8F%E5%91%B3&amp;action=edit&amp;redlink=1" class="new" title="意味 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>意味</rb><rp>(</rp><rt>의미</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E7%AD%89&amp;action=edit&amp;redlink=1" class="new" title="等 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>等</rb><rp>(</rp><rt>등</rt><rp>)</rp></ruby></a>을 <a href="/index.php?title=%E6%8A%8A%E6%8F%A1&amp;action=edit&amp;redlink=1" class="new" title="把握 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>把握</rb><rp>(</rp><rt>파악</rt><rp>)</rp></ruby></a>하기 어렵게 한다.
</p><p>새로운 <a href="/index.php?title=%EA%B8%80%E5%AD%97&amp;action=edit&amp;redlink=1" class="new" title="글字 (없는 문서)">글<ruby class="hanja grade70 unknown"><rb>字</rb><rp>(</rp><rt>자</rt><rp>)</rp></ruby></a>를 만드는 데에 있어서 가장 효율이 좋은 <a href="/index.php?title=%E9%81%8A%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="遊法 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>遊法</rb><rp>(</rp><rt>유법</rt><rp>)</rp></ruby></a>이기 때문에 이렇게 만든 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>의 80% <a href="/index.php?title=%E4%BB%A5%E4%B8%8A&amp;action=edit&amp;redlink=1" class="new" title="以上 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby></a>이 여기에 <a href="/index.php?title=%E5%B1%AC&amp;action=edit&amp;redlink=1" class="new" title="屬 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby></a>한다. <a href="/index.php?title=%E7%8F%BE%E4%BB%A3&amp;action=edit&amp;redlink=1" class="new" title="現代 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>現代</rb><rp>(</rp><rt>현대</rt><rp>)</rp></ruby></a>에 들어 <a href="/index.php?title=%E4%B8%AD%E5%9C%8B&amp;action=edit&amp;redlink=1" class="new" title="中國 (없는 문서)"><ruby class="hanja grade80 unknown"><rb>中國</rb><rp>(</rp><rt>중국</rt><rp>)</rp></ruby></a>에서 만들어진 <a href="/index.php?title=%E5%85%83%E7%B4%A0&amp;action=edit&amp;redlink=1" class="new" title="元素 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>元素</rb><rp>(</rp><rt>원소</rt><rp>)</rp></ruby></a>를 나타내는 글자들<sup id="cite_ref-1" class="reference"><a href="#cite_note-1">[1]</a></sup>도 이 <a href="/index.php?title=%E6%96%B9%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="方法 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby></a>으로 만들어졌다.
</p><p><a href="/wiki/%E9%9F%93" title="韓"><ruby class="hanja grade80"><rb>韓</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E8%AA%9E" title="語"><ruby class="hanja grade70"><rb>語</rb><rp>(</rp><rt>어</rt><rp>)</rp></ruby></a>, <a href="/wiki/%E6%BC%A2" title="漢"><ruby class="hanja grade72"><rb>漢</rb><rp>(</rp><rt>한</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E7%B5%90&amp;action=edit&amp;redlink=1" class="new" title="結 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>結</rb><rp>(</rp><rt>결</rt><rp>)</rp></ruby></a> 등 <ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby>의 <a href="/index.php?title=%E5%A4%A7%E9%83%A8%E5%88%86&amp;action=edit&amp;redlink=1" class="new" title="大部分 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>大部分</rb><rp>(</rp><rt>대부분</rt><rp>)</rp></ruby></a>이 여기에 <ruby class="hanja grade40"><rb>屬</rb><rp>(</rp><rt>속</rt><rp>)</rp></ruby>한다.
