<h2><span id="槪要(개요)"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_4&amp;action=edit&amp;section=1" title="부분 편집: 槪要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>아이폰 4는 <a href="/index.php?title=%EC%95%A0%ED%94%8C&amp;action=edit&amp;redlink=1" class="new" title="애플 (없는 문서)">애플</a>이 <a href="/index.php?title=2010%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2010年 (없는 문서)">2010<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a><a href="/index.php?title=6%E6%9C%887%E6%97%A5&amp;action=edit&amp;redlink=1" class="new" title="6月7日 (없는 문서)">6<ruby class="hanja grade80 unknown"><rb>月</rb><rp>(</rp><rt>월</rt><rp>)</rp></ruby>7<ruby class="hanja grade80 unknown"><rb>日</rb><rp>(</rp><rt>일</rt><rp>)</rp></ruby></a> <ruby class="hanja grade62"><rb>發表</rb><rp>(</rp><rt>발표</rt><rp>)</rp></ruby>한 <a href="/wiki/%EC%8A%A4%EB%A7%88%ED%8A%B8%ED%8F%B0" title="스마트폰">스마트폰</a>이다. <a href="/wiki/%EC%95%84%EC%9D%B4%ED%8F%B0" title="아이폰">아이폰</a>의 네<ruby class="hanja grade60"><rb>番</rb><rp>(</rp><rt>번</rt><rp>)</rp></ruby>째 <ruby class="hanja grade42"><rb>製品</rb><rp>(</rp><rt>제품</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="特徵(특징)"></span><span class="mw-headline" id=".E7.89.B9.E5.BE.B5.28.ED.8A.B9.EC.A7.95.29"><ruby class="hanja grade32"><rb>特徵</rb><rp>(</rp><rt>특징</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_4&amp;action=edit&amp;section=2" title="부분 편집: 特徵(특징)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_(1%E4%B8%96%E4%BB%A3)&amp;action=edit&amp;redlink=1" class="new" title="아이폰 (1世代) (없는 문서)">아이폰 (1<ruby class="hanja grade62 unknown"><rb>世代</rb><rp>(</rp><rt>세대</rt><rp>)</rp></ruby>)</a>부터 이어져 온 <a href="/index.php?title=%E6%B5%81%E7%B7%9A%E5%9E%8B&amp;action=edit&amp;redlink=1" class="new" title="流線型 (없는 문서)"><ruby class="hanja grade20 unknown"><rb>流線型</rb><rp>(</rp><rt>유선형</rt><rp>)</rp></ruby></a> 디자인을 버리고 <ruby class="hanja grade40"><rb>刻</rb><rp>(</rp><rt>각</rt><rp>)</rp></ruby>진 <ruby class="hanja grade72"><rb>一名</rb><rp>(</rp><rt>일명</rt><rp>)</rp></ruby> <a href="/index.php?title=%ED%86%B5%EC%A1%B0%EB%A6%BC&amp;action=edit&amp;redlink=1" class="new" title="통조림 (없는 문서)">통조림</a> 모양의 디자인이 되었다. <ruby class="hanja grade70"><rb>前後面</rb><rp>(</rp><rt>전후면</rt><rp>)</rp></ruby>은 <ruby class="hanja grade0"><rb>琉璃</rb><rp>(</rp><rt>유리</rt><rp>)</rp></ruby>로 덮여 있고, <ruby class="hanja grade32"><rb>側面</rb><rp>(</rp><rt>측면</rt><rp>)</rp></ruby>은 <a href="/index.php?title=%ED%94%8C%EB%9D%BC%EC%8A%A4%ED%8B%B1&amp;action=edit&amp;redlink=1" class="new" title="플라스틱 (없는 문서)">플라스틱</a> <ruby class="hanja grade52"><rb>材質</rb><rp>(</rp><rt>재질</rt><rp>)</rp></ruby>에 <a href="/index.php?title=%E9%87%91%E5%B1%AC&amp;action=edit&amp;redlink=1" class="new" title="金屬 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>金屬</rb><rp>(</rp><rt>금속</rt><rp>)</rp></ruby></a> 띠를 둘러 놨는데, 이 <ruby class="hanja grade40"><rb>金屬</rb><rp>(</rp><rt>금속</rt><rp>)</rp></ruby> 띠가 <a href="/index.php?title=%EC%95%88%ED%85%8C%EB%82%98&amp;action=edit&amp;redlink=1" class="new" title="안테나 (없는 문서)">안테나</a> <ruby class="hanja grade32"><rb>役割</rb><rp>(</rp><rt>역할</rt><rp>)</rp></ruby>을 한다.
</p><p><ruby class="hanja grade60"><rb>畵面</rb><rp>(</rp><rt>화면</rt><rp>)</rp></ruby>은 3.5인치로 크기는 <ruby class="hanja grade52"><rb>以前</rb><rp>(</rp><rt>이전</rt><rp>)</rp></ruby> <ruby class="hanja grade62"><rb>世代</rb><rp>(</rp><rt>세대</rt><rp>)</rp></ruby>와 같지만, <a href="/index.php?title=%EB%A0%88%ED%8B%B0%EB%82%98_%EB%94%94%EC%8A%A4%ED%94%8C%EB%A0%88%EC%9D%B4&amp;action=edit&amp;redlink=1" class="new" title="레티나 디스플레이 (없는 문서)">레티나 디스플레이</a>가 <ruby class="hanja grade40"><rb>適用</rb><rp>(</rp><rt>적용</rt><rp>)</rp></ruby>되어 <a href="/index.php?title=%E8%A7%A3%E5%83%8F%E5%BA%A6&amp;action=edit&amp;redlink=1" class="new" title="解像度 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>解像度</rb><rp>(</rp><rt>해상도</rt><rp>)</rp></ruby></a>가 두<ruby class="hanja grade50"><rb>倍</rb><rp>(</rp><rt>배</rt><rp>)</rp></ruby>인 640x960이 되었다.
</p><p><a href="/index.php?title=2011%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="2011年 (없는 문서)">2011<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>에는 ‘화이트’ <ruby class="hanja grade52"><rb>色相</rb><rp>(</rp><rt>색상</rt><rp>)</rp></ruby> 모델도 출시되었다.
</p>
<h2><span id="論難(논란)"></span><span class="mw-headline" id=".E8.AB.96.E9.9B.A3.28.EB.85.BC.EB.9E.80.29"><ruby class="hanja grade42"><rb>論難</rb><rp>(</rp><rt>논란</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%EC%95%84%EC%9D%B4%ED%8F%B0_4&amp;action=edit&amp;section=3" title="부분 편집: 論難(논란)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p>안테나 <ruby class="hanja grade32"><rb>役割</rb><rp>(</rp><rt>역할</rt><rp>)</rp></ruby>을 하는 <ruby class="hanja grade40"><rb>金屬</rb><rp>(</rp><rt>금속</rt><rp>)</rp></ruby> 띠 <ruby class="hanja grade62"><rb>部分</rb><rp>(</rp><rt>부분</rt><rp>)</rp></ruby>을 <ruby class="hanja grade60"><rb>特定</rb><rp>(</rp><rt>특정</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>方式</rb><rp>(</rp><rt>방식</rt><rp>)</rp></ruby>으로 잡을 <ruby class="hanja grade40"><rb>境遇</rb><rp>(</rp><rt>경우</rt><rp>)</rp></ruby> <ruby class="hanja grade60"><rb>通信</rb><rp>(</rp><rt>통신</rt><rp>)</rp></ruby> <a href="/index.php?title=%E4%BF%A1%E8%99%9F&amp;action=edit&amp;redlink=1" class="new" title="信號 (없는 문서)"><ruby class="hanja grade60 unknown"><rb>信號</rb><rp>(</rp><rt>신호</rt><rp>)</rp></ruby></a>가 잡히지 않는 <ruby class="hanja grade62"><rb>問題</rb><rp>(</rp><rt>문제</rt><rp>)</rp></ruby>가 <ruby class="hanja grade52"><rb>發見</rb><rp>(</rp><rt>발견</rt><rp>)</rp></ruby>되었다.
</p>
