<h2><span id="槪要"></span><span class="mw-headline" id=".E6.A7.AA.E8.A6.81"><a href="/wiki/%E6%A7%AA%E8%A6%81" title="槪要"><ruby class="hanja grade32"><rb>槪要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></a></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97&amp;action=edit&amp;section=1" title="부분 편집: 槪要">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E5%85%AD%E6%9B%B8" title="六書"><ruby class="hanja grade62"><rb>六書</rb><rp>(</rp><rt>육서</rt><rp>)</rp></ruby></a>의 분류 중 하나로 두 개 이상의 <a href="/wiki/%E6%BC%A2%E5%AD%97" title="漢字"><ruby class="hanja grade70"><rb>漢字</rb><rp>(</rp><rt>한자</rt><rp>)</rp></ruby></a>와 그 뜻을 합쳐서 만든 글자를 가리킨다.
</p>
<h2><span id="특징"></span><span class="mw-headline" id=".ED.8A.B9.EC.A7.95">특징</span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%9C%83%E6%84%8F%E6%96%87%E5%AD%97&amp;action=edit&amp;section=2" title="부분 편집: 특징">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><a href="/wiki/%E8%B1%A1%E5%BD%A2%E6%96%87%E5%AD%97_(%E6%BC%A2%E5%AD%97)" title="象形文字 (漢字)"><ruby class="hanja grade40"><rb>象形</rb><rp>(</rp><rt>상형</rt><rp>)</rp></ruby></a>이나 <a href="/wiki/%E6%8C%87%E4%BA%8B%E6%96%87%E5%AD%97" title="指事文字"><ruby class="hanja grade42"><rb>指事</rb><rp>(</rp><rt>지사</rt><rp>)</rp></ruby></a> 같은 <a href="/index.php?title=%E6%96%B9%E6%B3%95&amp;action=edit&amp;redlink=1" class="new" title="方法 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>方法</rb><rp>(</rp><rt>방법</rt><rp>)</rp></ruby></a>으로 나타낼 수 있는 <a href="/index.php?title=%E6%A7%AA%E5%BF%B5&amp;action=edit&amp;redlink=1" class="new" title="槪念 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby></a>이 <a href="/index.php?title=%E9%99%90%E7%95%8C&amp;action=edit&amp;redlink=1" class="new" title="限界 (없는 문서)"><ruby class="hanja grade42 unknown"><rb>限界</rb><rp>(</rp><rt>한계</rt><rp>)</rp></ruby></a>에 다다르자 둘 <a href="/index.php?title=%E4%BB%A5%E4%B8%8A&amp;action=edit&amp;redlink=1" class="new" title="以上 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>以上</rb><rp>(</rp><rt>이상</rt><rp>)</rp></ruby></a>의 글자를 합쳐서 새로운 <ruby class="hanja grade32"><rb>槪念</rb><rp>(</rp><rt>개념</rt><rp>)</rp></ruby>을 나타내기 <a href="/index.php?title=%E5%A7%8B%E4%BD%9C&amp;action=edit&amp;redlink=1" class="new" title="始作 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>始作</rb><rp>(</rp><rt>시작</rt><rp>)</rp></ruby></a>한 <a href="/index.php?title=%E7%B5%90%E6%9E%9C&amp;action=edit&amp;redlink=1" class="new" title="結果 (없는 문서)"><ruby class="hanja grade52 unknown"><rb>結果</rb><rp>(</rp><rt>결과</rt><rp>)</rp></ruby></a>이다.
</p><p><a href="/index.php?title=%E4%BF%A1&amp;action=edit&amp;redlink=1" class="new" title="信 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>信</rb><rp>(</rp><rt>신</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E6%9E%97&amp;action=edit&amp;redlink=1" class="new" title="林 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>林</rb><rp>(</rp><rt>임</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E6%98%8E&amp;action=edit&amp;redlink=1" class="new" title="明 (없는 문서)"><ruby class="hanja grade62 unknown"><rb>明</rb><rp>(</rp><rt>명</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E4%BC%91&amp;action=edit&amp;redlink=1" class="new" title="休 (없는 문서)"><ruby class="hanja grade70 unknown"><rb>休</rb><rp>(</rp><rt>휴</rt><rp>)</rp></ruby></a> 등이 여기에 속한다.
