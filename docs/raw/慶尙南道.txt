<h2><span id="概要(개요)"></span><span class="mw-headline" id=".E6.A6.82.E8.A6.81.28.EA.B0.9C.EC.9A.94.29"><ruby class="hanja grade0"><rb>概要</rb><rp>(</rp><rt>개요</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%85%B6%E5%B0%99%E5%8D%97%E9%81%93&amp;action=edit&amp;section=1" title="부분 편집: 概要(개요)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><b><ruby class="hanja grade32"><rb>慶尙南道</rb><rp>(</rp><rt>경상남도</rt><rp>)</rp></ruby></b>는 <a href="/wiki/%E5%A4%A7%E9%9F%93%E6%B0%91%E5%9C%8B" title="大韓民國"><ruby class="hanja grade80"><rb>大韓民國</rb><rp>(</rp><rt>대한민국</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade72"><rb>道</rb><rp>(</rp><rt>도</rt><rp>)</rp></ruby>이다.
</p>
<h2><span id="歷史(역사)"></span><span class="mw-headline" id=".E6.AD.B7.E5.8F.B2.28.EC.97.AD.EC.82.AC.29"><ruby class="hanja grade52"><rb>歷史</rb><rp>(</rp><rt>역사</rt><rp>)</rp></ruby></span><span class="mw-editsection"><span class="mw-editsection-bracket">[</span><a href="/index.php?title=%E6%85%B6%E5%B0%99%E5%8D%97%E9%81%93&amp;action=edit&amp;section=2" title="부분 편집: 歷史(역사)">편집</a><span class="mw-editsection-bracket">]</span></span></h2>
<p><ruby class="hanja grade42"><rb>高宗</rb><rp>(</rp><rt>고종</rt><rp>)</rp></ruby>33<ruby class="hanja grade80"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby>(<a href="/index.php?title=%E8%A5%BF%E7%B4%80&amp;action=edit&amp;redlink=1" class="new" title="西紀 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>西紀</rb><rp>(</rp><rt>서기</rt><rp>)</rp></ruby></a><a href="/index.php?title=1896%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1896年 (없는 문서)">1896<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>)에 <a href="/index.php?title=%E6%85%B6%E5%B0%99%E9%81%93&amp;action=edit&amp;redlink=1" class="new" title="慶尙道 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>慶尙道</rb><rp>(</rp><rt>경상도</rt><rp>)</rp></ruby></a>가 <ruby class="hanja grade32"><rb>慶尙南道</rb><rp>(</rp><rt>경상남도</rt><rp>)</rp></ruby>와 <a href="/wiki/%E6%85%B6%E5%B0%99%E5%8C%97%E9%81%93" title="慶尙北道"><ruby class="hanja grade32"><rb>慶尙北道</rb><rp>(</rp><rt>경상북도</rt><rp>)</rp></ruby></a>로 나뉘었다.
</p><p><ruby class="hanja grade52"><rb>以後</rb><rp>(</rp><rt>이후</rt><rp>)</rp></ruby> <a href="/index.php?title=1963%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1963年 (없는 문서)">1963<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>, <a href="/index.php?title=%E9%87%9C%E5%B1%B1%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="釜山市 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>釜山市</rb><rp>(</rp><rt>부산시</rt><rp>)</rp></ruby></a>가 <a href="/index.php?title=%E9%87%9C%E5%B1%B1%E7%9B%B4%E8%BD%84%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="釜山直轄市 (없는 문서)"><ruby class="hanja grade10 unknown"><rb>釜山直轄市</rb><rp>(</rp><rt>부산직할시</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade32"><rb>昇格</rb><rp>(</rp><rt>승격</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade62"><rb>分理</rb><rp>(</rp><rt>분리</rt><rp>)</rp></ruby>되었고, <a href="/index.php?title=1995%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1995年 (없는 문서)">1995<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a> <a href="/index.php?title=%E6%A2%81%E5%B1%B1%E9%83%A1&amp;action=edit&amp;redlink=1" class="new" title="梁山郡 (없는 문서)"><ruby class="hanja grade32 unknown"><rb>梁山郡</rb><rp>(</rp><rt>양산군</rt><rp>)</rp></ruby></a>의 <ruby class="hanja grade62"><rb>一部</rb><rp>(</rp><rt>일부</rt><rp>)</rp></ruby>가 <a href="/index.php?title=%E6%A9%9F%E5%BC%B5%E9%83%A1&amp;action=edit&amp;redlink=1" class="new" title="機張郡 (없는 문서)"><ruby class="hanja grade40 unknown"><rb>機張郡</rb><rp>(</rp><rt>기장군</rt><rp>)</rp></ruby></a>으로 <ruby class="hanja grade40"><rb>分離</rb><rp>(</rp><rt>분리</rt><rp>)</rp></ruby>되어 <a href="/wiki/%E9%87%9C%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="釜山廣域市"><ruby class="hanja grade12"><rb>釜山廣域市</rb><rp>(</rp><rt>부산광역시</rt><rp>)</rp></ruby></a>에 <ruby class="hanja grade32"><rb>編入</rb><rp>(</rp><rt>편입</rt><rp>)</rp></ruby>되었다.
</p><p><a href="/index.php?title=1997%E5%B9%B4&amp;action=edit&amp;redlink=1" class="new" title="1997年 (없는 문서)">1997<ruby class="hanja grade80 unknown"><rb>年</rb><rp>(</rp><rt>년</rt><rp>)</rp></ruby></a>에는 <a href="/index.php?title=%E8%94%9A%E5%B1%B1%E5%B8%82&amp;action=edit&amp;redlink=1" class="new" title="蔚山市 (없는 문서)"><ruby class="hanja grade12 unknown"><rb>蔚山市</rb><rp>(</rp><rt>울산시</rt><rp>)</rp></ruby></a>가 <a href="/wiki/%E8%94%9A%E5%B1%B1%E5%BB%A3%E5%9F%9F%E5%B8%82" title="蔚山廣域市"><ruby class="hanja grade12"><rb>蔚山廣域市</rb><rp>(</rp><rt>울산광역시</rt><rp>)</rp></ruby></a>로 <ruby class="hanja grade32"><rb>昇格</rb><rp>(</rp><rt>승격</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade40"><rb>分離</rb><rp>(</rp><rt>분리</rt><rp>)</rp></ruby>되어 <ruby class="hanja grade30"><rb>只今</rb><rp>(</rp><rt>지금</rt><rp>)</rp></ruby>에 이른다.
</p>
